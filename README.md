# 📊 Superstore Sales Dashboard | Microsoft Excel

An interactive **Sales Analytics Dashboard** built entirely in **Microsoft Excel** using Pivot Tables, Pivot Charts, Slicers, Conditional Formatting, and Excel formulas. The dashboard transforms raw Superstore sales data into meaningful insights, helping users analyze sales performance, customer behavior, and product profitability.

---


# 📌 Project Overview

This project demonstrates how **Microsoft Excel** can be used to build a complete Business Intelligence dashboard without relying on Power BI or other visualization tools.

The dashboard enables users to monitor overall sales performance, analyze customer distribution, evaluate product profitability, and explore trends through interactive filters and dynamic visualizations.

# 📈 Dashboard Insights

The dashboard helps users answer important business questions such as:

- How do sales change throughout the year?
- Which regions generate the highest sales?
- Which states contribute the most to revenue?
- Which states have the largest customer base?
- Which customer segments place the most orders?
- Which states generate high sales but low profit?
- What is the average discount offered in each state?
- Which product categories contribute the most to sales?
- Which product categories generate the highest profit?
- Which products deliver the highest profit?
- How are quantities distributed across product categories and subcategories?
